Nuestra Historia

NEXUS Gaming nació en 2024 con una misión clara: revolucionar la gestión de inventarios en tiendas de videojuegos.

Desarrollado con las últimas tecnologías web, nuestro sistema combina potencia, simplicidad y eficiencia para ofrecerte una experiencia de gestión sin precedentes.

Construido con Spring Boot 3, MySQL 8 y las mejores prácticas de desarrollo empresarial, NEXUS Gaming es la solución que tu negocio necesita.

Misión

Proporcionar herramientas de gestión accesibles y potentes para negocios del sector gaming.

Visión

Ser la plataforma líder en gestión de inventarios de videojuegos en Latinoamérica.

Valores

Innovación, calidad, seguridad y compromiso con nuestros usuarios.

Tecnologías Utilizadas

Backend

Java 21, Spring Boot 3.2, Spring Data JPA, Spring Security

Frontend

Thymeleaf, Bootstrap 5, JavaScript ES6+

Base de Datos

MySQL 8.0

Seguridad

Spring Security 6, BCrypt, JWT
