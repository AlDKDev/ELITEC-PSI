NEXUS Gaming es un sistema de gestión de inventarios especializado en tiendas de videojuegos. Permite controlar stock, precios, categorías y generar reportes en tiempo real.

El sistema está construido con:
  • Backend: Java 21, Spring Boot 3.2
  • Frontend: Thymeleaf, Bootstrap 5
  • Base de Datos: MySQL 8.0
  • Seguridad: Spring Security 6
